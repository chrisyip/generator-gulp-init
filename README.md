# Gulp.js Generator

[![NPM version][npm-image]][npm-url] [![Dependency Status][daviddm-url]][daviddm-image]

[Gulp.js](http://gulpjs.com/) generator for [Yeoman](http://yeoman.io).

### Usage

Install with npm:

```
npm install -g generator-gulp-init
```

After install, run:

```
yo gulp-init
```
